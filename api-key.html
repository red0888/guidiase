How to get sms-activate API key

Step-1:

Visit the website at https://sms-activate.ru/en/ , it should look something like this:

Step-2:

Create an account or login by accessing the top right panel.

Step-3:

Now, you have to add funds to your wallet. To do so, click the top right icon and select "Top-up balance".

Step-4:

Select your preferred mode of payment from the list, choose an option that is supported in your country. The amount should appear in your wallet in rubles after a successful payment. (Refer above image)

Step-5:

In order to get your API key, select the "info" tab as shown below, and click on "API" in the submenu.

Step-6:

Now you can get your API key from the box. Copy the API key and paste it in Xyliase, and you're good to go!

Note: You can view the pricing of phone number of each country at the home page. The amount charged for making each token will depend on the country you choose in Xyliase.
